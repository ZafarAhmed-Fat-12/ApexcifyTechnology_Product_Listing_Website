# Product Marketplace
This project is a dynamic and interactive Product Marketplace built using HTML, CSS, and JavaScript. It features a responsive design, product filtering, a featured product showcase, and functional cart and wishlist systems. The product data is fetched from a live API, providing a realistic e-commerce experience.
Features

Dynamic Product Listing: Fetches product data from dummyjson.com to display a variety of products.
Product Showcase: A prominent section to highlight a featured product with navigation controls.
Filtering and Search: Users can filter products by category, price range, and search by product title, brand, or category.
Pagination: Products are displayed with pagination for easy browsing.
Add to Cart Functionality: Users can add products to a shopping cart, with an updated counter.
Wishlist Functionality: Users can add products to a wishlist, with an updated counter and persistent storage using localStorage.
Responsive Design: The layout adapts to different screen sizes for a seamless user experience on desktop and mobile devices.
Modern UI/UX: Clean and attractive design with smooth animations and interactive elements.

# Technologies Used
HTML5: For structuring the web content.
CSS3: For styling and responsive design.
JavaScript (ES6+): For dynamic content, interactivity, and API integration.
Font Awesome: For icons.

# Setup Instructions
To get a copy of this project up and running on your local machine, follow these simple steps:

git clone <repository-url>
# 2 Navigate to the project directory:
cd product-marketplace
3 Open index.html: Simply open the index.html file in your web browser. All necessary CSS and JavaScript files are linked locally.

# Usage
Use the category and price filters to narrow down product listings.
Type in the search bar to find specific products.
Click on any product card to view it in the featured showcase.
Use the left and right arrows in the showcase to navigate through featured products.
Click "Add to Cart" or "Add to Wishlist" to interact with the product.
The cart and wishlist counters at the top left will update dynamically.
